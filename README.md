# wish-you-tag
Wish you... be happy coding! An online 'wish you...' text generator.

## Why this?
I saw a status on my Facebook news feed which look something like this:
```
┏┷ ┓
┃ ข ┃
┃ อ ┃
┃ ใ ┃
┃ ห้ ┃
┃ . ┃
┃ . ┃
┃ . ┃
┃ . ┃
┃ . ┃
┗━ ┛
```

...What? That's all!
